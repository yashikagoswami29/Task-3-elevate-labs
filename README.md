# Task 3: SQL for Data Analysis

## Objective
To use SQL queries to extract and analyze data from a database.

## Tools Used
- MySQL 8.0
- MySQL Command Line Client

## Dataset
Created a sample Ecommerce database with Customers, Products, and Orders tables.

## SQL Operations Performed
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- Aggregate Functions (SUM, AVG)
- Subqueries
- Views
- Indexing

## Outcome
Learned how to manipulate, query, and analyze structured data using SQL.
